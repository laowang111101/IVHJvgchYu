# 基于Python的膳食健康系统 python962

## 项目概述

本项目是基于Python语言的膳食健康系统，结合了前端Vue框架和后端Django框架，使用Mysql作为数据库，旨在提供全面的食物营养成分数据库和个性化的健康分析。

## 技术栈

- 后端：Python 3.7.7
- 前端：Vue
- 数据库：Mysql 5.7
- 后端框架：Django
- 开发工具：PyCharm
- 数据库可视化工具：Navicat

## 系统角色

本系统主要包括以下角色：

- 用户：注册、登录、管理个人信息
- 系统管理员：负责系统管理和维护

## 运行环境

- 操作系统：Windows 10 / MacOs

## 功能模块

### 用户模块

- 用户注册、登录
- 个人信息管理

### 食物类别与成分模块

- 食物营养成分数据库
- 包含热量、蛋白质、脂肪等关键指标

### 食谱信息模块

- 食谱分类展示
- 烹饪步骤及营养分析

### 膳食记录模块

- 每日饮食记录
- 摄入量计算与可视化报告

### 健康分析模块

- 饮食结构合理性评估
- 个性化改善建议

### 互动社区模块

- 论坛分类与互动问答
- 健康饮食知识共享
